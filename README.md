# Heart Disease Visual Explorer

## Overview
The **Heart Disease Visual Explorer** is an interactive Python-based dashboard that explores heart disease risk factors using advanced visualizations.  
It allows users to understand the relationships between lifestyle, medical features, and heart disease probability with intuitive charts, plots, and density visualizations.

---

## Features
- **Interactive Charts:** Explore relationships between BMI, sleep patterns, cholesterol, and more.
- **Feature-Based Visualizations:** Focus on medically relevant features to identify risk factors.
- **Clean Dark Theme:** Modern, sleek dark UI with subtle glows and glassy cards.
- **Minimal Interactivity:** Hover effects and skill highlights without overwhelming the interface.
- **Customizable:** Easy to extend with additional features or datasets.
